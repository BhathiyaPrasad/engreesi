# exit

🗣️ /ˈeɡzɪt/

## Sinhala Meaning(s)

### පිටවෙනවා

## Usage

### ~ (adv./prep.)

- He asked for his father's permission to exit the room.


### ~ from something

- He asked for an exit from the crowded room.


### ~ something

- She asked for the exit code from the security guard.
